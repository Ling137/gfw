“VPN Gate”是2013年横空出世的翻墙工具，并且在2014年1月开放源代码（正式成为“开源项目”）。
它采用“分布式”的方法提供 VPN Server，以此来对抗 GFW 的“IP 黑名单”。
很多 VPN Gate 的 Server 是海外网友志愿提供的。如果你担心 Server 本身的安全性，建议采用 VPN + TOR 的方式构造“双重代理”。
用了“VPN + TOR”的双重代理之后，即使是 Server 的管理者也【无法】看到你通过该 Server 的真实上网流量。


官方网站是 https://www.vpngate.net/cn/
（必须翻墙才能访问）
为了方便大伙儿下载，它的官网提供了每日更新的镜像网站列表。
这些镜像可以【免翻墙】访问，让你用来下载安装包。

由于 VPN Gate 的安装包比较大，而且它的官网已经提供了上述【免翻墙镜像】。
所以，俺就不在 BT Sync 中提供 VPN Gate 的安装包了。


当你能翻墙之后，建议到它官网的如下网址，进行邮件订阅。
https://www.vpngate.net/cn/mail.aspx
订阅之后，你的邮箱会自动收到“上述镜像网址”的每日更新。
